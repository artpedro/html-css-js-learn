Tag Main

Na tag main estará todo o conteúdo original desta página, ou seja, não inclui headers, footers, nav e tags que estão presentes em outras páginas

Logo, essa tag não deve ser descendente de outras tags como footer, header, nav, aside ou article

Tag Article

A tag article define um artigo com um assunto específico dentro de uma páginas. Ela deve ser construida de forma que faça sentido ao ser lida isoladamente do site, ou seja, deve conter um assunto independente e completo

Tag Header

Essa tag está sendo utilizada no início desse documento. Trata-se de um cabeçalho, que deve conter as informações e links introdutórios do documento em questão. Geralmente, nela estará presente a barra de pesquisa, logo do site, tag Nav para navegação no site e outras tags de carácter estrutural ou introdutório.

Nav

Nav é uma tag usada para navegação no site. Geralmente contém links para areas diferentes do site.

Tag Aside

Essa tag é utilizada para explicar ou completar informações a cerca de um elemento ou termo utilizado em algum texto. Ela está sendo utilizada acima para explicar a Tag nav.

Tag Footer

Essa tag é define o rodapé da página em questão. Dentro dela, geralmente está contidas informações de contato, nomes dos autores, direitos autorais, links de navegação. Informações de contato e endereço devem ser incluídas dentro da tag Address que costuma acompanhar a tag Footer.

Tag Section

Essa tag nada mais é que uma Tag genérica. Evite utiliza-la quando já existe uma tag apropriada para a função que ela está cumprindo.
