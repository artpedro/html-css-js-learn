Tag main

Na tag main estará todo o conteúdo original desta página, ou seja, não inclui headers, footers ou nav, tags que estão presentes em outras páginas


Logo, essa tag não deve ser descendente de outras tags como footer, header, nav, aside ou article

nav

nav é uma tag usada para navegação no site. Geralmente contém links para areas difereentes do site.
