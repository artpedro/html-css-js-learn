Dados pessoais:

'Text': Cria uma caixa de texto que permite ao usuário inserir dados em formato de texto. É a tag mais comum usada em formulários.

'Email': Cria uma caixa de texto para receber um email, o navegador faz uma simples verificação para checar se o email é válido.

'Date': Cria um campo que permite ao usuário inserir uma data, geralmente usado para campos que requerem uma data específica.

'Radio': Cria um botão de opção que permite ao usuário selecionar apenas uma opção de uma lista de opções.

'Option': Um tipo de diferente de input, este cria uma caixa de texto que abre diversas opções de resposta ao clickar.

'Checkbox': Cria uma caixa de seleção que permite ao usuário selecionar uma ou mais opções de uma lista.

'Color': Cria uma caixa com uma cor para o usuário selecionar a cor desejada no gradiente.

'File': Cria uma caixa de seleção que permite ao usuário selecionar um ou mais arquivos para serem enviados para o servidor.

'Number': Cria uma caixa de texto que permite ao usuário inserir dados numéricos, geralmente usado para campos que requerem entrada numérica.

'Password': Cria uma caixa de texto que oculta o texto digitado, geralmente usado para campos de senha.

'Password': Cria uma caixa de texto de tamanho flexível para o usuário escrever textos maiores.

'Url': Cria uma caixa de texto para o usuário digitar um link/url. O navegador faz uma simples verificação para assegurar que é um url.
